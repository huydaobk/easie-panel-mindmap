# EASIE Sandwich Panel Mindmap

Interactive mindmap visualization of EASIE FP7 Project research on sandwich panels.

**Live demo:** https://huydaobk.github.io/easie-panel-mindmap

## About

This mindmap contains comprehensive knowledge about sandwich panels compiled from:
- EASIE FP7 Project (Grant No. 213302)
- BS EN 14509-2013 European Standard
- European Recommendations for Sandwich Panels
- Design-by-Testing guidelines

## Structure (v2.0 - Refactored)

The mindmap is organized into **13 logical groups (A-M)** following engineering design workflow:

| Group | Topic | Focus |
|-------|-------|-------|
| **A** | Fundamentals & System Overview | Entry point, taxonomy, panel types |
| **B** | Materials & Component Properties | Faces, cores, bonds, properties |
| **C** | Standards, Design Basis & Validation | EN 14509, DBC vs DBT, safety |
| **D** | Structural Behaviour & Failure Mechanisms | Loads, behaviour, failure modes, ULS/SLS |
| **E** | Joints, Fasteners & Interface Detailing | Connections, stiffness, waterproofing |
| **F** | Openings, Penetrations & Discontinuities | Cut-outs, reinforcement, verification |
| **G** | Application-Specific Design | Wall, roof, cladding, partitions |
| **H** | Frameless / Structural Shell Buildings | Load-bearing panels, diaphragms, FE |
| **I** | Durability, Ageing & Long-Term Performance | Environment, creep, property reduction |
| **J** | Fire, Safety & Robustness | Reaction, resistance, detailing |
| **K** | Manufacturing, Handling, Installation & QA/QC | FPC, tolerances, site inspection |
| **L** | Worked Examples, Design Aids & Workflow | Examples, formulas, checklist |
| **M** | Reference Library / Evidence Map | Standards, deliverables, sources |

## Key Improvements (v2.0)

- **Logical workflow**: Follows engineer's thinking from fundamentals → materials → standards → behaviour → details → applications
- **Clear separation**: Domain knowledge vs. support artifacts (formulas, checklist, references)
- **Cross-link ready**: Failure modes, durability, openings, joints as "cross-cutting" topics
- **EASIE-aligned**: Reflects project focus on frameless buildings, DBT vs DBC, ageing, cladding
